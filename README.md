# muddy

A mud client in your browser usin' node.js & websockets.

## Installation & Usage

    $ npm install websocket-server
    $ git clone git://github.com/dannytatom/muddy.git
    $ cd muddy
    $ vim config.js
    $ node server.js &
    $ chromium client/index.html
